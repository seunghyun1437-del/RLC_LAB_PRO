# 프로젝트 제목: RLC 공진·임피던스 시각화 웹앱

## 📌 개요
브라우저에서 R, L, C, f, Vin을 입력하면 **공진주파수(f₀), Q, 대역폭(BW, f₁·f₂)**를 계산하고  
**임피던스 |Z(f)| 또는 전류 |I(f)|** 주파수 응답을 그래프로 직관적으로 보여주는 웹앱입니다.

## ⚙️ 주요 기능
- R/L/C/f/Vin 입력 → f₀, Q, BW(–3 dB), ζ 계산 및 결과 카드 표시
- 그래프 **토글**: **|Z(f)| ↔ |I(f)|** (공진에서 |I| 피크 확인)
- **공진 시각화**: f₀ 세로선, –3 dB 경계(f₁·f₂) 및 대역 음영
- **그래프 PNG 저장** 버튼(보고서/제출용)

## 🛠 기술 스택
- HTML, CSS, JavaScript (Vanilla)
- Chart.js (CDN 활용)

## 🚀 실행 방법
1. `index.html`을 브라우저로 엽니다.  
   - *참고: Content Delivery Network를 사용하므로 인터넷 연결이 필요합니다.*
2. 원하는 R/L/C/f/Vin 값을 입력 후 계산하기 버튼을 누르고 결과를 확인합니다.

## ▶️ 사용 방법
1. 좌측 **파라미터**에 R[Ω], L[H], C[F], f[Hz], Vin[V] 입력 후 **계산하기** 클릭  
2. 하단 그래프에서 **|Z| / |I|** 버튼으로 보기 전환  
3. 공진 구간(–3 dB, f₁~f₂)과 f₀ 마커를 확인  
4. 필요하면 **🖼️그래프 PNG 저장** 버튼으로 이미지 저장
